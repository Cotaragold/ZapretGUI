Performance Optimization:
   ✅ No frame-based animations (only trigger-based)
   ✅ SnapsToDevicePixels enabled everywhere
   ✅ Effects only on hover/selected (not permanent except Card)
   ✅ Smooth 60fps target on modern hardware

════════════════════════════════════════════════════════════════════════
ANIMATION SETTINGS:
════════════════════════════════════════════════════════════════════════

Button Hover Scale Animation:
   • Storyboard: ButtonHoverScaleIn / ButtonHoverScaleOut
   • Target Property: RenderTransform.ScaleX & ScaleY
   • From: 1.0 → To: 1.02
   • Duration: 150ms (0:0:0.15)
   • Trigger: IsMouseOver=True (enter), False (exit)
   • RenderTransformOrigin: 0.5, 0.5 (center)

No Animations:
   ✅ TextBox, ListBox, TabControl (only color changes)
   ✅ Animations are minimal (no constant loops)
   ✅ Only scale on Button for subtle feedback

════════════════════════════════════════════════════════════════════════
USAGE EXAMPLES:
════════════════════════════════════════════════════════════════════════

BUTTON:
───────
<Button Content="Primary Button"
        Style="{StaticResource NeonPrimaryButtonStyle}" />
<Button Content="Secondary Button"
        Style="{StaticResource NeonSecondaryButtonStyle}" />
<Button Content="Danger"
        Style="{StaticResource NeonDangerButtonStyle}" />

TEXTBOX:
────────
<TextBox Text="Enter text..."
         Style="{StaticResource NeonTextBoxStyle}" />

LISTBOX (Note: Include ItemContainerStyle!):
──────────────────────────────────────────────
<ListBox Style="{StaticResource NeonListBoxStyle}"
         ItemContainerStyle="{StaticResource NeonListBoxItemStyle}">
    <ListBoxItem Content="Item 1" />
    <ListBoxItem Content="Item 2" />
</ListBox>

TABCONTROL (Note: Style each TabItem!):
─────────────────────────────────────────
<TabControl Style="{StaticResource NeonTabControlStyle}">
    <TabItem Header="Tab 1"
              Style="{StaticResource NeonTabItemStyle}">
        <TextBlock Text="Content 1" />
    </TabItem>
    <TabItem Header="Tab 2"
              Style="{StaticResource NeonTabItemStyle}">
        <TextBlock Text="Content 2" />
    </TabItem>
</TabControl>

CARD:
─────
<Border Style="{StaticResource NeonCardStyle}">
    <StackPanel>
        <TextBlock Text="Card Title" />
        <TextBlock Text="Card content" />
    </StackPanel>
</Border>

CUSTOMIZATION POINTS:
════════════════════════════════════════════════════════════════════════

Easy to Customize:

1. Colors: Edit NeonColors.xaml
   <Color x:Key="NeonCyan">#FF00FFFF</Color>

2. Glow Intensity: Edit DropShadowEffect in NeonControls.xaml
   <DropShadowEffect BlurRadius="14" Opacity="0.7" />

3. Animation Speed: Edit Storyboard Duration
   <DoubleAnimation Duration="0:0:0.15" />

4. Sizes: Edit CornerRadius in templates
   <Border CornerRadius="12" />

5. Padding: Edit Padding properties
   <Setter Property="Padding" Value="12,8" />
